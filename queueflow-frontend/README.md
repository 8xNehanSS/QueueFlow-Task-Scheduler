# QueueFlow Frontend

QueueFlow Frontend is the client application for the QueueFlow project. It provides the user interface for interacting with queue-related workflows.

## Features

- Modern frontend application structure
- QueueFlow user interface
- Ready for backend API integration
- Easy to extend and customize

## Prerequisites

- Node.js
- npm, yarn, or pnpm

## Getting Started

1. Install dependencies:

	```bash
	npm install
	```

2. Start the development server:

	```bash
	npm run dev
	```

3. Open the local URL shown in the terminal.
